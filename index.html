Overview

What iTreeView does

iTreeView extracts properties from CAD models and displays them in a structured tree view so engineers can review assemblies, locate components, and generate documentation more efficiently.

Structured data extraction

Pulls useful component data into an organized and readable tree.

Visual assembly understanding

Shows parent-child relationships in a clear graphical format.

Documentation tools

Exports BOM and canvas information into formats that are easier to share.

Feature Overview

Core application functions

Built to simplify CAD assembly review, property extraction, and communication.

Inventor Imports

Load from Open IAM

  • Connects to the currently open CAD assembly.
  • Extracts assembly structure and component properties.
  • Builds the initial tree structure used by iTreeView.

Refresh

  • Re-reads the CAD assembly and updates the tree.

Properties

  • Allows the user to select the properties to extract from CAD.
  • Metadata can include:
Part Number Description Material Quantity Mass Assembly Mass with/without Ref Mass Mass Overridden Assembly Relationships Custom Properties

Note: Some properties may not work as expected during beta production of the application.

Inventor Actions

Select in CAD

  • Highlights the selected component directly in the CAD model.
  • Helps quickly locate parts within large assemblies.

Zoom to Component

  • Automatically zooms the CAD view to the selected component.

Open File

  • Opens the source part or subassembly file in CAD.
Export Tools

Export BOM to Excel

  • Exports the TreeView BOM with all selected properties to Excel.

Export Canvas Image

  • Generates a copy of the Canvas Tree View into Excel.
  • Includes:
Hierarchical Structure Part Number Description QTY

Future Updates

Selectable Properties for Each Node Node Color Selection Export to Visio
Search

Search Box

  • Instantly locate parts by part number, description, or name.
  • Highlights matching components in the tree.
Canvas View

The main canvas displays the assembly structure as an intuitive visual tree where assemblies appear as parent nodes and subassemblies and parts branch underneath.

Each node can contain:

Component Image Part Number Description

This allows engineers to understand complex assemblies at a glance.
